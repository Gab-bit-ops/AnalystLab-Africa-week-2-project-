# 🎬 Netflix Content Analysis
# 📌 Project Overview
This project explores Netflix's content library to uncover trends in content distribution, ratings, genres, and geographic availability. Data cleaning was performed in MySQL and insights were visualized using Power BI.
Internship: Analyst Lab Africa — Week 2
# 🛠️ Tools & Technologies
Tools:  MySQL,  Purpose: Data ingestion & cleaning
Tools: Power BI, Purpose: Dashboard & visualization
# 🧹 Data Cleaning Process
✅ Imported 9K+ rows via LOAD DATA INFILE into a structured schema
✅ Checked for duplicates using show_id as primary key — zero duplicates found
✅ Audited all 12 columns for NULL and empty values
✅ Replaced missing entries with 'Unknown' across key fields: director, cast, country, rating, duration, description

# 🔍 Key Insights
🇺🇸 USA dominates content volume with 2.8K titles; India follows at 1.0K
🔞 TV-MA is the #1 rating at 3.2K titles — Netflix heavily targets mature audiences
📅 Content peaked in 2018–2020 (1,030–1,145 titles/year) before a notable decline
🎭 Dramas & International Series are the most listed genre categories
🎬 Movies outnumber TV Shows 2:1 (6K vs 3K)
